有时添加 add-apt-repository出错

cd /etc/apt/sources.list.d

安装sublime

菜单栏居然在最上面，奇怪。。（害我重装一次）

取消更新提示

“Preferences”=> “Settings-User” 进入个人参数设置页面；

点击菜单－help－Enter License，输入：

sublime中文设置

在https://packagecontrol.io/installation下载插件 Package Control.sublime-package

打开Preferences菜单，并选择 Browse Packages，系统会打开Packages文件夹，回到上一级菜单，然后打开Installed Packages文件夹

将下载的压缩包放进去

重启sublime，选择 Preferences/Package Control 菜单的Install Package准备安装中文字符包

在搜索框中输入utf，搜索到对应模块ConvertToUTF8，点击安装，系统会自动从网上下载并安装

http://blog.sina.com.cn/s/blog_4ca83f830102wxg2.html

中文输入

Linux环境下Sublime不能输入中文

2克隆项目到本地 :

git clone https://github.com/lyfeyaj/sublime-text-imfix.git

3运行脚本 :

cd sublime-text-imfix && ./sublime-imfix

4 重启 sublime 就可以中文输入

12.美化

安装numix图标：

安装Unity管理工具，这是一个神器，可以用来设置主题、图标等等：

sudo apt-get install unity-tweak-tool
把Ubuntu的启动器位置改为和Windows一样在桌面底部显示：

gsettings set com.canonical.Unity.Launcher launcher-position Bottom
如果要还原成默认的可以执行：

gsettings set com.canonical.Unity.Launcher launcher-position Left
